## 🔗 API Resource Guide

### 🏠 Public Access
| Method | Endpoint | Action |
| :--- | :--- | :--- |
| `POST` | `/api/register` | Initialize new system account |
| `POST` | `/api/login` | Authenticate and generate Bearer token |

### 🔒 Protected Resources
> Header Requirement: `Authorization: Bearer {token}`

<details>
<summary><b>👤 User Management</b></summary>
<br>

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| `GET` | `/api/users` | List all system entities |
| `POST` | `/api/users` | Register internal staff |
| `PUT` | `/api/users/{id}` | Update account metadata |
| `DELETE` | `/api/users/{id}` | Terminate account access |
</details>

<details>
<summary><b>🎓 Scholarship Programs</b></summary>
<br>

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| `GET` | `/api/scholarships` | View all available grants |
| `POST` | `/api/scholarships` | Launch new scholarship program |
| `PUT` | `/api/scholarships/{id}` | Update program requirements |
| `DELETE` | `/api/scholarships/{id}` | Archive scholarship program |
</details>

<details>
<summary><b>📝 Applicant Tracking</b></summary>
<br>

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| `GET` | `/api/applicants` | Review scholarship queue |
| `POST` | `/api/applicants` | Process new student application |
| `PUT` | `/api/applicants/{id}` | Modify applicant information |
| `DELETE` | `/api/applicants/{id}` | Remove record from system |
</details>
